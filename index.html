Conhecimentos

HTML5

HTML (HyperText Markup Language) é a linguagem de marcação usada para estruturar páginas da web. Ele define elementos como títulos, parágrafos, imagens, links e botões, permitindo a criação de sites.

CSS3

CSS3 (Cascading Style Sheets 3) é uma linguagem de estilos usada para definir a aparência de páginas HTML. Ele permite personalizar cores, fontes, espaçamentos, animações e layouts, tornando os sites mais bonitos e responsivos.

JavaScript

JavaScript é uma linguagem de programação voltada para a criação de interatividade em páginas web. Com ele, é possível manipular elementos HTML, responder a eventos do usuário, criar animações, validar formulários e até desenvolver aplicações completas no frontend e backend.

Node.Js

Node.js é uma plataforma que permite executar JavaScript no servidor, facilitando a criação de aplicações rápidas e escaláveis. É amplamente utilizado para desenvolver servidores web, APIs, e aplicações em tempo real.

React

React é uma biblioteca JavaScript para criar interfaces de usuário interativas e dinâmicas, utilizando componentes reutilizáveis. Ele permite construir aplicações rápidas e eficientes com atualizações rápidas por meio do DOM virtual. Ideal para aplicações de página única (SPA).

TypeScipt

TypeScript é uma linguagem de programação baseada em JavaScript, mas com tipagem estática. Ela adiciona tipos ao código, o que ajuda a detectar erros antes da execução, tornando o desenvolvimento mais seguro e organizado. É compilada para JavaScript, garantindo compatibilidade com qualquer ambiente.

Bootstrap

O Bootstrap é um framework front-end que facilita a criação de sites responsivos e modernos, oferecendo um sistema de grid, estilos prontos e componentes interativos como botões e modais. Ele agiliza o desenvolvimento e melhora a aparência das interfaces.

Git Hub

O GitHub é uma plataforma de hospedagem de código que usa Git para controle de versão. Ele permite colaboração, gerenciamento de projetos e armazenamento de repositórios, facilitando o desenvolvimento e o trabalho em equipe.

Atenticação JWT

A autenticação JWT (JSON Web Token) é um padrão para autenticação e autorização, que permite que um servidor verifique a identidade de um usuário sem precisar armazenar informações sobre ele. O JWT funciona como um "passe digital", contendo informações sobre o usuário (como permissões e identificação) de forma codificada e assinada.
